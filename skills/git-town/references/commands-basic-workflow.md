# Commands: basic workflow

Quick help:
- Overview: `git town`
- Detailed help: `git town help <command>`

Day-to-day commands:
- `git town hack` - create a new feature branch off main
- `git town sync` - update the current branch with ongoing changes
- `git town switch` - switch between branches interactively
- `git town propose` - open a pull/merge request for the current branch

Typical flow:
- `git town hack` -> `git town sync` -> `git town propose` -> `git town ship`
