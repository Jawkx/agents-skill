# Commands: setup and config

Setup and configuration:
- `git town init` - setup assistant
- `git town config` - show/update Git Town config
- `git town config get-parent` - show parent branch
- `git town config remove` - remove Git Town config from repo
- `git town completions` - generate shell completions
- `git town offline` - enable/disable offline mode
