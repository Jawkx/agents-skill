# Commands: stacked changes

Commands for managing branch stacks and lineage:
- `git town append` - create a child branch
- `git town prepend` - insert a branch between current and parent
- `git town commit` - commit staged changes into an ancestor branch
- `git town set-parent` - change a branch parent
- `git town swap` - swap branch with its parent
- `git town up` - switch to the parent branch
- `git town down` - switch to a child branch
- `git town diff-parent` - show changes vs parent
- `git town merge` - merge current branch with its parent
- `git town detach` - move a branch out of a stack
- `git town walk` - run a command across branches
