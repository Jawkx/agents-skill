# Commands: branch type and sync control

Use these to change how a branch syncs by setting its type.

Branch type commands:
- `git town contribute` - mark a branch as contribution
- `git town observe` - mark a branch as observed
- `git town park` - park a branch (no syncing)
- `git town prototype` - mark a branch as prototype (no push)
- `git town feature` - mark a branch as feature

Related:
- `git town hack` also creates or converts to a feature branch.
