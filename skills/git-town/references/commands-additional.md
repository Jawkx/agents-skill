# Commands: additional

Supporting commands:
- `git town branch` - show branch hierarchy and types
- `git town compress` - squash a branch into one commit
- `git town delete` - delete a feature branch
- `git town rename` - rename a branch
- `git town repo` - open repo in browser
- `git town ship` - deliver a completed branch
- `git town help` - help about any command
